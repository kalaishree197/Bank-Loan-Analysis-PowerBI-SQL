# 🏦 Bank Loan Report - Power BI Dashboard

*Interactive Multi-Page Power BI Dashboard with SQL Analysis*

A professional data analytics project focused on analyzing bank loan portfolio performance using SQL for data processing and Power BI for interactive visualization and reporting.

---

## 📋 Project Overview

This project demonstrates end-to-end financial data analysis for a bank’s loan dataset. It includes:

- SQL queries for calculating key performance indicators (KPIs), good vs bad loan segmentation, month-over-month trends, and various aggregations.
- A clean, interactive *Power BI dashboard* consisting of *3 pages* designed for clear insights into loan performance, credit risk, and business trends.

The dashboard was built by following a structured YouTube tutorial, with custom enhancements for better usability and storytelling.

---

## 🛠 Tech Stack

- *SQL* – Data extraction, KPI calculations, trend analysis, and loan categorization (Good vs Bad Loans)
- *Microsoft Power BI* – Interactive dashboard development, DAX measures, visualizations, and slicers

## 🧠 Key Business Insights

The analysis provides several actionable insights that can guide strategic lending decisions, risk management, and operational improvements:

- *Portfolio Quality: Approximately **86%* of the loans are classified as *Good Loans* (Fully Paid + Current), while *Bad Loans* (Charged Off) account for around *14%*. This indicates generally healthy repayment behavior, but the Bad Loans segment still poses a notable credit risk that warrants targeted mitigation strategies.

- *Interest Rate & Default Risk*: Higher interest rates show a strong positive correlation with loan defaults. Reducing exposure to high-interest, high-risk grades or implementing stricter pricing adjustments can help lower default rates and improve overall portfolio returns.

- *Loan Purpose Analysis*: Debt consolidation and credit card refinancing are the most popular loan purposes. Focused marketing campaigns, simplified application processes, and tailored product offerings for these segments can drive higher application volumes while maintaining risk controls.

- *Borrower Stability*: Applicants with longer employment lengths (especially 10+ years) exhibit significantly better repayment performance. Credit scoring models and approval policies can be refined to give more weight to employment tenure, potentially reducing future defaults.

- *Geographic Concentration*: A large portion of loan applications and funded amounts comes from a few key states (e.g., California, New York, Texas). Banks can optimize regional marketing budgets, branch staffing, and localized risk assessment models accordingly.

- *Temporal & Recovery Trends*: Month-over-month and MTD comparisons reveal seasonal patterns in loan issuance and repayments. These trends support better cash flow forecasting, liquidity planning, and early intervention in collections for underperforming segments.

- *Recovery Opportunities*: The gap between funded amounts and amounts received highlights inefficiencies in the recovery process for Bad Loans. Strengthening collection strategies and early delinquency management can improve recovery rates and reduce losses.

These insights enable data-driven decisions in credit policy, customer acquisition, risk pricing, and operational efficiency — ultimately supporting healthier loan growth with controlled risk.
